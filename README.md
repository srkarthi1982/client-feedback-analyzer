# Client Feedback Analyzer

Client Feedback Analyzer is an Ansiversa mini-app focused on collecting, organizing, and summarizing client feedback.

## V1 Capabilities

- Capture batch/project context and review notes
- Add, edit, and delete feedback items
- Filter by sentiment, category, and priority
- Draft key strengths, concerns, and recommended actions
- Generate live structured summary preview
- Copy summary output and full report output
- Persist draft locally in browser storage

## Local Development

```bash
npm install
npm run typecheck
npm run build
npm run dev
```
